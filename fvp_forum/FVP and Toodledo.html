Gadgets,

I keep my FVP list in Toodledo sorted by Date Added. That way all new tasks are added to the bottom of the list. I use the Star feature of Toodledo to mark the tasks I am going to work on. For tasks that I work on but don't complete, I first clone the task, which puts a copy of it at the bottom where it belongs (after a screen refresh) and then mark the original task as "done". If I want to track something I worked on in the task, I update the note in the original task before I clone it. This system has been working very well for me for quite some time.
