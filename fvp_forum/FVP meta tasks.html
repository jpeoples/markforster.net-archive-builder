This is just a short list of a few "meta" tasks that I gathered out of this forum and used in my own FVP.

- weed list
- stop working for today
- go to bed, sleep
- make a break of X minutes
- break / lunch
- break from working
- change question (the question you use to build the chain)
- change tool (another list, diary, timed routine etc)

Any others I've missed?
