How long do you work on each task on the list before postponing it and working on something else?

1. Do you work on it till you get bored with it?
2. Do you work on it for 10 minutes or some other predetermined duration?
3. Do you work on it till you have completed some number of smaller subtasks?
4. Do you use some other heuristic?

Mark:

Any of these are acceptable!

The basic principle is that you work on it until you feel you've done enough for the time being and it's time to move onto something else. This will obviously vary with the circumstances, whether you are just starting a difficult job, how urgent it is, etc.
