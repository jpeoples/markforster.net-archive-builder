Christopher:

Have I got this right? You don't want to do the first task and there's no other task on the list which you want to do more than the first task. Therefore there is not a single task on your to-do list which you want to do. So in effect you are suffering from a general paralysis.

It happens sometimes.

Some possible remedies are:

- Think up some tasks which you do want to do, and add them to the list, AND/OR

- Work on the first task for five minutes and no more (use a timer). Do the same for the second task if you're still stuck - and so on, AND/OR

- Take a timed break of at least 30 minutes (an hour would be better) in which you can do anything you like as long as it isn't work, AND/OR

- If this is a frequent occurrence examine whether you are doing the right sort of work for you.
