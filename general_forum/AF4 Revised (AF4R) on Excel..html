There is a way to adapt AF4R digitaly and it seems be working quiet well event if I still prefer paper.

1. Take an excel sheet
2. Name the colums
- Project > Here goes your project's name
- Category > Dissmiss, New, old, Recurring, Tickler, Unfinished,
- Description > Here describes the task
- When > I put the month here or W49 (week number 49)
- Day > Monday, tuesday and so on
- Duration

3 during the day sort your colums by day, projects , categories, duration

4. If something is dissmiss you can color it in yellow

5. If something is unfinished if you need the answer today color it it green

6. If something is finished don't care about the past - erase it ! But if you need an historical report it on your project.


It's quiet fast and easy to use.
