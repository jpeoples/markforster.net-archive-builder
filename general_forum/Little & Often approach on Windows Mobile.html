Hi Roger: Yes, I always open my file direct from the SD card onto the main PC but ... I keep it in a folder My Mobile Dropbox (along with other things) which I regularly synchronize (with SyncToy) to a similarly named folder in My Dropbox on the main PC, which gives me main PC and internet backups just in case!

Using the file stored on the SD card is noticably slower at times, but it is quite adequate. Before disconnecting the smartphone, I always click the "Safely remove hardware" button in the taskbar as a precaution. While away from my PC I use the file on the smartphone.

Re doing a mass change, I found one can do it via Excel. Highlight all the records in ListPro, copy and paste into a blank Excel sheet. Do search and replace (or whatever), then copy the rectangle of cells in Excel, highlight the last row in ListPro and paste back.

If you are changing the table structure in ListPro, do so before copying and pasting to Excel.

Hope this helps :-)

Roger:

The My Mobile Dropbox folder is just my "hack" to ensure my list gets backed up to the Net. There is no Dropbox for Windows Mobile (although there is for the iPhone). So SynchToy shunts my file to the desktop within my DropBox folder, which gets backed up to the net automatically.

I did try using Allway Sync for a while, which works in the background synchronizing every time a file changes. But I found it did not play nice with open ListPro files, so it was back to SynchToy.

Not using OneNote. Just a very simple ListPro list.

Re ActiveSync, I am using Microsoft Exchange to synch calendar and contacts with Google, it works with my old phone but I have heard there are problems with WM6.5. Not really using ActiveSync directly to the desktop any more.
