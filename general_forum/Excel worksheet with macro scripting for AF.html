Hi,

I created an Excel worksheet with some macro scripting to support AF.
This is what's implemented by now:

- It supports the creation of multiple notebooks in one file. Each worksheet is a notebook; each worksheet column is a list (page).

- It lets the user to define the amount of lines per list (page), will show exactly that amount in the worksheet (by hiding the rest), and won't let the user to input anything past it. It also allows redefining this amount whenever needed, and will rearrange the lists accordingly. For example: If a notebook was set to 30 lines per list, and later re-arranged to 15 lines per list, all the existing tasks will be redistributed to 4 pages.

- It supports shortcuts to "Complete", "Dismiss" and "Pause" a task. This last one crosses off the task, and re-adds it at the end of the last active page. It also allows re-opening a task that's been dismissed or completed (though this shouldn't be used).

- It monitors each list, and will show them dimmed when all of their tasks are either completed or dismissed.

- It lets the user to jump to the first active page, and to add a task at the end of the last active page.

Shortcuts:

CTRL-Shift-A: Re-arranges a list (Asking lines/page)
CTRL-Shift-N: Creates a new task at the end of the last active page
CTRL-Shift-C: Crosses off a task as completed
CTRL-Shift-D: Marks a task as dismissed (with yellow background)
CTRL-Shift-P: "Pauses" a task (crosses-it off, re-adds it at the end)
CTRL-Shift-O: Restores a task to active status
CTRL-Shift-F: Jump to the first active page

Frankly, I created this until I decide if I want to split my tasks into work/home. It apparently works ok (I only started using it yesterday), though the feeling is that AF works somehow better on paper, as many say. (I made it more or less look like paper, anyway :)

So, for whoever's interested, even out of curiosity, here's the link for downloading:
http://rapidshare.com/files/212885404/AF.xls.html
