I had been using MLO since the start of the beta, but found the Windows Mobile version laborious (to copy and paste a partially done item to the last page of my outline was taking several keystrokes).

So I'm switching to ListPro, which I haven't used for years. I'm just using it on the PDA, which I can view when at a PC on the larger screen with MyMobile (most standard PC keystrokes apply). It seems more suitable for AF. I have a folder AutoFocus with Home, Office etc files, which load instantly. Drag and drop works, and lines are numbered automatically so one knows when to start a new page/list. Dismissed tasks can be highlighted. I set up a list template called Page, and the program automatically named the ensuing lists Page, Page 2, Page 3, etc (which was a nice surprise).

Issues to do with whether to have one or multiple AF lists have been among my main concerns, and I feel these are now largely resolved with ListPro, as the speed means there is negligible overhead to pasting the same task to more than one file.

The only thing I miss from MLO - and I really miss it - is the facility to get a neat list of things done today. But the same would be true of pen and ink - which is also very tempting!

Hope this helps someone.
