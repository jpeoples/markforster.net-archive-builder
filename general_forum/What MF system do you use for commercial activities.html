Alison:

If you are dealing with a number of clients, then it is important to keep client documentation. When I was coaching clients, I kept a file on each client and made a lot of use of standard forms. Any notes referring to a client went in my In-Tray so I could file them as part of my daily routine. That way it's easy to find where you are with each client.

Also make use of project management tasks such as "Review Client X", "Update Client Files", etc.
