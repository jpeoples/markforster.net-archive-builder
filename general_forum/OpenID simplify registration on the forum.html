Registration on Mark Forster's forum takes time from him and users, which is why i thought OpenID a good solution. OpenID is a recent technology that allows you to have one login and to log in to every site that supports OpenID. For example you have an OpenID provider, say, Livejournal account, you enter username.livejournal.com, your password and you are logged in.

I don't know, how it is possible to implement OpenID on this site, but if it is possible, it would be great. Everyone, who has an OpenID (that includes everyone, who has Google, Yahoo, Livejournal, Wordpress.com, Livejournal accounts), could easily login without registration, and if somebody doesn't, there are plenty of OpenID providers.

You can find more about it here:
http://openid.net/

The list of OpenID providers can be found here:
http://openid.net/get-an-openid/

P.S. No, this is not spam, i really love Autofocus, and this post is a wish to improve this forum. :)

Yes, spammers can easily create OpenIDs, everyone can. You can use one of the services sindikat mentioned or even install you own if you want. I have my own OpenID server running on my website, and it was as easy as installing a simple php script.

That's not to say OpenID is a bad idea in itself, it just doesn't solve this forum's problem. Using OpenID for login could work, but only in conjunction with either an automatic spam filtering mechanism and/or a moderation team. And managing all of that is probably more trouble for Mark than manually creating accounts now and then.
