So I am using checklists for routines in the morning and at noon today and at night time. Also I am using a no list where I write down what I need to do for the next hour or so. I'm using a blank index card 5x8 and attaching it to a clipboard of that size. That seems to be working out well. There are usually some items that I don't get to. Usually I throw the card away. All of this reduces greatly the number of pages I use. However it is not so great for working on projects. It handles mostly routine items. I have tried to to vote one page for each project. With some success.
I do check back to this forum everyday but if I have been commenting a lot I feel like it will be better for me to wait for others to post.
