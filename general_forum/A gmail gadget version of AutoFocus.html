Gmail has a "task" add-on that would work ok. I like how when you check something off it draws a line through it and then kind of greys it out so the stuff to still be done is easy to read. You can move items up or down (instead of re-writing them) and indent if needed (for sub tasks). One problem is the closed-list aspect of AF. Since there's no "pages" you don't get that. A work around would be using the "new list" feature. Make a different task list when you reach a certain number of items, like 25-30 or so and that would close the lists for you. It may work for some people, but I get too much satisfaction physically crossing stuff off with a pen.

If anyone tries it out I'd like to hear how it goes for them.

Topper:

Google docs is an interesting possibility especially as it's accessible via an API. At the moment there doesn't seem to be any API for accessing gmail tasks.

For tables in google docs a google spreadsheet would be the way to go I suspect.

There could be a column for the task name, a column for a status character (empty=not started, . = started like a dot in AF not book, x=done, d=dismissed perhaps). All that would be need after that would be separator rows between the 'pages' which would be groups of rows. Finished pages could be purged by just deleting groups of rows. I think it's possible to do colouring and formatting but that would probably be too time consuming.

There are applications for accessing google documents and spreadsheets on mobile devices but I don't know how well they work.
