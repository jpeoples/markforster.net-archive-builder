Since you're not now utilizing the context field, what if you did? Name it something like "money owed", then exclude it from your main work view.

I'm using OmniFocus quite happily with multiple contexts, even though contexts are not officially part of DWM.

My main OF perspective is Grouping=DUE, Sorting=DUE. This gives me a long list, from today on into the future, regardless of any assigned context. You could do this, select ALL contexts, unselect the one you don't want to see, then save the result as a new perspective.

You'd probably want to add a task: check "money owed" context!
