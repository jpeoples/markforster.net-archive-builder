I have searched literally for years for the "Best" application
to use for time management, GTD and now AfxDwm. I have
tried just about everything and have come across some
very good and well featured programs.

But every application, no matter how well it seemed to work,
left me with a feeling of constriction and claustrophobia.

I yearned to breathe free, to break out of the golden cage.

I have come to the conclusion, digitally, the best solution may be to
use no application at all. Just use the file system.

Which means, for example, using folder titles for AF entries
Then one can use any and all applications or no applications
for your system achieving maximum flexibility.

I arrived at this conclusion after cleaning up my computer
desktop according to the excellent advice found in this
article.

http://www.lifeclever.com/how-to-organize-your-cluttered-desktop-and-regain-your-sanity/

I thought If you can follow their system based on GTD why not use it for autofocus.

Probably not an original idea but there it is.

One very useful feature of this approach is that not only
can each folder contain files and data connected with the
action but actions can be linked with any files folders, other actions or
locations on your hard drive through the use of the aliasing
function.

Can such a system be exported? Yes. One possiblity is
zipping it which should preserve the folder structure at
least to a few levels but perhaps not the aliases.
