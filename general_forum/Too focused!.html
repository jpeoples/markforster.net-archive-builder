First of all, thanks to Mark for setting up this beta and creating this system.

A little background: I was a floundering GTDer for a couple of years with my procrastination and its resistance was preventing me from fully embracing the GTD methodology. (I was NEVER able to complete a "weekly review"). Perhaps the New Year inspires us to refresh, so I started at the GTD forums looking for ideas and saw others discussing DIT. I then found Mark's site just in time for his introduction of AF. I signed up for the Beta and received his intro email after I got home on Tuesday. After kids were asleep, I started developing my list and ended up staying up until 1am.

So far I have 99 items on my 4 page work list with 32 crossed off and only one "dismissed" so far.

The amazing thing has been my focus. Somehow picking the task that stands out has so little resistance that I easily gain momentum and stay focused for a longer time than before when I was working on a "next action". It seems like the task was like a boulder on the top of a hill and just starting working on it starts it rolling down and its momentum is difficult to stop. Sometimes that's a good thing. Sometimes not, like if I have something I need to complete today, but I've been on such a roll on a different task that I lose track of time.

So what I started doing is jotting a timestamp next to my task of when I started. If I feel I'm rolling downhill too long I can do a quick check to see how long I've been working on this task. Since I started it, most of the time, it's more than 30 minutes. When I stop, I jot an endtime. This way I'm starting to get a handle on how long I'm working on things which I suppose could be useful in estimating work load in the future.

I'll check in again next week to see if any of the "new-ness" excitement has worn off.

-John

If you want a timer against tasks, you could try Abstract Spoon's ToDoList2 http://www.codeproject.com/KB/applications/todolist2.aspx ); many more options than you probably want. Free and Open Source. Show only the columns you want. If you want to use it to contain AF, you could make it a page to a database - and all open databases are shown on separate tabs so it is very easy to move from one to the other. For each task you start the timer when you start and stop when you stop and it keeps a running total; you can edit that total manually if you forget to start/stop it sometimes. Always assumes you are happy having the PC running all the time.
