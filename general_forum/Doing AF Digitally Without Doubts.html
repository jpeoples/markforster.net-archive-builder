I have settled on using Toodledo for my AF lists (accessing it through Appigo's ToDo on the IPhone). One thing about my approach is very AF-like. I can choose how to approach my tasks, rather than feeling obligated to use pure AF. I think my approach could be very helpful to those who are waffling and having a hard time deciding if AF is for them.

Because I am using aspects of other productivity methods in my system, I no longer feel that constant uncertainty. For example, I tag all my errands with @errands. I don't always use it, but if I want to be GTD in my approach, I'm easily able to. I really like using a goal-directed approach, so I've tagged all my tasks that relate to my ten goals for the year with the appropriate goal. I like using due dates to guide me, too, so every task that has a firm deadline has that added to it. Finally, I love the feeling of accomplishment I got with DIT and other approaches. Every day I can look on my Toodledo statistics page and see how my productivity compares to previous days. I especially love that I can see if I am doing more tasks than I'm adding so my number of outstanding tasks is decreasing. I also just added a goal to work through every page a day. I didn't make it today, but Toodledo will add a link to a chain every time I do accomplish that goal. I will try to see how many days in a row I can accomplish that without breaking the chain. Adding this information takes very little time and is done as I review my tasks each evening.

Using a fuller-featured program like Toodledo for AF allows you to leave features unused, but also to begin using some of them if your circumstances change or you're just in need of a change. So often I think change is all we need to kick up our productivity. Doing AF digitally allows you to tweak the system occasionally without completely abandoning the approach. Hope that is helpful to someone.

Just thought I'd pop this in: I got an iPod touch, and have been trying out Autofocus digitally. I love paper and pen - I agree it has a feel nothing can replace - but when using the notebook, I found I had too many tasks in one place. My assignments for the quarter (I am in naturopathic medical school) were in Things; the weekly reading list is enormous; my partner and I are raising kids; I have lots of recurring tasks; and I needed to be able to "see" them all in one list. If I have project planning one place and a task list another place and a list somewhere of what else has to be done then it doesn't work.

Why didn't I just enter them? First, I found project planning on paper is not feasible for me. Adding steps when a page is filled is a mess. Second, I have horrible handwriting. I can't read it at times. Third, there are LOTS.

So I've played with various apps on the iPod. I've enjoyed features of both ToDo and Toodledo (I found Things didn't work quite as I would like) I'd like to put in a word for the Toodledo app. It has great recurring task settings (e.g. I can put in for every two months if need be, can't do that on ToDo). I can star tasks for a "today" list. It has wonderful sorting settings.

And with a pro account, you get... subtasks. For me, this is essential. It lets me put in an entry about someone's birthday (which I can make recurring) and then make some notes about what I need to do for that - I am notorious for forgetting birthdays and sending lame cards with elephants on them. Or it lets me put in an assignment, well ahead of time, and then add steps as needed.

Granted, it doesn't sync with my iCal, and I'm not much on web lists. But it does very well as a standalone app, and I manage my lists on my iPod, because sometimes I'm in front of the computer, sometimes I'm not. (And then there are all the medical apps coming up for it...)

But I'm still carrying the little Moleskine volant!
