Hi Jupiter!

My current AF4 set up is very simple. Rejected items get the highlighter treatment.

Page divided down the middle: left = AF4 Entry, right = notes area

AF4 entries are by Keyword in capitals (e.g. FITNESS) with a Sub Root for recall / catagorisation in lower case (e.g 10K run Chesterfield).

Written thus: FITNESS / 10K run Chesterfield

I write all my notes in the same A4 Spiral bound notebook, starting from the back. On these pages the Keyword and Sub Root are written at the bottom of the page. I allocate a page per topic, where neccesary. When the page gets filled up it can be removed and placed in a file, if needed.

Credit to Gerry (Ultra Simple Guide to Time Management) for this idea.

Gerry describes the use of RAW in his method for note making (Result, Action, Waiting for) but I prefer the use of DAP (Data, Analysis, Plan) which is often used for recording health care interventions. If I'm waiting for something it goes onto my AF4 list if I need it within a few days. Otherwise it goes on my calander.

My files are labelled in the same way, e.g. FITNESS / 10K run Chesterfield

Works for me. And this is about as simple as I can make it.

Good luck!
