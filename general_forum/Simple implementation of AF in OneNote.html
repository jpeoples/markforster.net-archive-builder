Here is my simple implementation of AF4 (my "customised" version of it) in OneNote.

OneNote is very nice software to implement AF because of following features:

1) It's easy to integrate with Outlook (links, reminders, contacts)
2) It's very easy to move tasks from one list to another (e.g. from Open List to Closed List)
3) You can tag your notes and then collect (filter) notes by tags. For example - "call" or "shopping"
4) You can print page(s) if you decide to go "paper" or like editing notes in paper.
5) Very nice feature - OneNote remembers the time of creation/modification of each note, even in one single text block - very useful for DWM users (which I'm not).

I have created one text block with 4 separators - Current Initative, Urgent (Today) Actions, Closed List, Open List.

Current Initiative - this is from DIT, that is the single project which you do everyday, first, before any other actions to push this project forward.
Urgent Action - what is to be done today (like take someone home from hospital)

Closed List and Open List are from AF4 - notes can be easily dragged from one to another.
