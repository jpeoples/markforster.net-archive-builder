A 2 or 3 column table in Word should take care of DWM - today's date, expire date, and task description (plus any other columns that you need)

OTOH, if you have Excel, I think that might work better just because Excel's formulas for today, today+7, today+30, etc are (to me) easier to work with. But if you don't want/need the Excel formulas, Word can handle DWM.
