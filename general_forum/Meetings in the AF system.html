I thought it might be usefull to describe how I feel meetings can be integrated in the system. I do it via a sequence of actions.

First, I enter "Prepare meeting about xxx". Once I've finished my preparation (potentially through several rewritings of the preparation action), I enter an action "Process meeting notes".

After the meeting has taken place, I'll come across this entry, take out my notes and start going through them, creating an additional entry for each action point decided during the meeting. If I have to create the minutes, either I've done this at the same time, or else I write an additional action for it. So now I can file my notes, confident that everything that was decided during the meeting will actually get done.

What do others think? How do you use the system for meetings?
