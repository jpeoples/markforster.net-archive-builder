AutofocusAHK version 0.9.4 is out: http://cloud.github.com/downloads/andreashofmann/AutofocusAHK/AutofocusAHK-0.9.4.zip

I bumped the version number because of one new major feature: real-time search! Pressing Capslock + F will open a search dialog. Type part of a task name and watch the possibilites narrow down. At any time, double-click a task in the results list to jump there.

Since two people so far had problems with using and uninstalling the application: this is a portable application. Just unzip somewhere you have access rights (in Vista/7, this means *NOT* the Program Files folder!) and run AutofocusAHK.exe to start the program. The app saves all information in the same directory the exe file is and can even be run from an usb device or be synced via dropbox or similar services. To uninstall, just delete its folder, done. EXCEPT if you activated the "start with windows" option in the preferences. This will create a shortcut in your start menu's startup folder. If you just delete the app, it will try to find it everytime windows starts, but fail. To solve this, either delete the shortcut by and or first uncheck "start with windows" in the preferences and then delete the app.

Also, if you don't know how to use the app, make sure to read the included readme, or the github main page, or the original thread here in the forum: http://www.markforster.net/forum/post/869348

Hi Andreas, not sure whether it's the best way to do it. But I've modified my copy of DWM_GetWorkWndowTitle and GetCurrentMetaData to get expiry dates in caps-D Work window title bars such as ...

Work - **TODAY** - DWM - AutofocusAHK 0.9.4.2 SMS
Work - Tomorrow - DWM - AutofocusAHK 0.9.4.2 SMS
Work - 2010-02-23 - DWM - AutofocusAHK 0.9.4.2 SMS

... the modification to DWM_GetWorkWindowTitle below...

DWM_GetWorkWindowTitle()
{
  global TaskExpire
  GetCurrentMetadata()
  Expire := SubStr(TaskExpire, 1,10)
  FormatTime, Today,, yyyy-MM-dd
  Tomorrow := A_Now
  Tomorrow += 1, days
  FormatTime, Tomorrow, %Tomorrow%, yyyy-MM-dd
  If (Expire == Today)
    Expire := "**TODAY**"
  Else If (Expire == Tomorrow)
    Expire := "Tomorrow"
  Title := "Work - " . Expire
  Title .= GetStandardWindowTitle()
  Return Title
}
