Updates for 1 March 2009:
=======================================

Added review modes
--------------------------------------------------------
There is now a new menu item called "review" with the subitems "all", "active", "done" and "dismissed", which allows you to view your list without actually working on it. You can't mark tasks as done here, but you can re-add dismissed/done items or choose to never re-add them, which removes them from review view.

Added Option to display “Quick Add Task”
--------------------------------------------------------
Finally you don't have to go to the separate task add page to add tasks to your list, but you can choose to display a quick add bar above or below your list. This can be configured under "Settings".

Added Option to display day markers
--------------------------------------------------------
You can now choose between the old page creation date once on every page or day markers that appear within the list every time a new day started. This is also configurable under settings.

Re: Handwriting.

I've always had poor handwriting. I long ago given up long hand and went to lettering. Even so, I wrote in a crabbed style which was poorly spaced with letters of uneven size. Then I realized that if I just slowed down and put my focus on each letter I COULD do better. And it worked. Now it may seem like a big sacrifice to slow down, it is not. The payback I get from being able to quickly read what I wrote (rather than having to decipher it) gives me that time back each time I read my writing ... which is more times than I wrote it ;-)

I suggest just trying it for a day or so and seeing if you have the same results. Form each letter carefully and keep the vertical size and horizontal spacing even. Lather, rinse, repeat ;-)

2009-06-9 11:36
Derek:

You can't dismiss single tasks. Af.cc tracks whether you already did something on the current page on the current pass. If you click "next page" when you are not allowed to do so by the rules, the app will tell you that and offer you to dismiss all remaining tasks on that page - if you use short captions, it will be a red button labeled "D!".

An exception to this is the last page, which you can always leave without dismissing.
