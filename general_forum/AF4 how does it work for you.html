While I have set up project books with lists of tasks, I also have a daily pocketbook - Moleskine pocket notebook - which constitutes my Closed List, again to keep some focus going. I use the RH page, one per day, the LH page is for expenses, quick and dirty notes, new tasks if my main list book is not close by, even my daily Internet usage in MB because I have a monthly cap where I'm living and working.

More later, suffice to say that I have managed to set up my current AF4 using ADOCs stationery due to its flexibility.

Jupiter has combined his project list and AF4 task list. All the projects are in AF4. If you don't want a project to get lost, I guess you must simply make sure to work on it rather than dismiss it.

As for next actions, you simply write with the project.

Build deck - draw up blueprint.

And when you have your blueprint drawn, you rewrite the project on the end of list

Build deck - get prices for materials

So this is not GTD. It's working on one task at a time, and rewriting it as needed.

I do basically the same thing, but I have a binder and one page is "Build deck" and on that page is "draw blueprint" (eventually that blueprint will also be in the binder, or a "Build deck" folder ) and "get prices for materials".

2010-10-7 11:08
Agree with Mark in principle, but contexts can help as long as one can restrict the number of contexts.

GTD, MLO etc turned me off with contexts just being one of the many features that caused system overhead, manual or digital.

Due to my peripatetic work/life-style, I use a limited number of contexts (which I put on separate lists usually, not as tags on the main list, the main list contains the triggers as noted in a previous post above). I cannot be in two places in the short-term, let alone simultaneously.

So I use 'Travel' as a context as sometimes an airport is the next place I can get a decent enough Internet connection to be able to file online eg my VAT return (like sales tax for you guys across the Pond).

If I had a long daily commute where someone else did the driving, eg on a train, I'd probably include that as a context.

My checklists include the next flight (tends to be 5 - 11 hours or more door-to-door) because if I forget something, it can be a real problem.

MY UK taxi driver ALWAYS asks all his clients for the airport, 'Got your passport?' but my travel checklist for carryon/cabin contains all the IT items, files etc, and a pair of underpants, because ' a fool and his underwear are soon parted.'

I might share my thoughts on specific checklists if anyone is interested.

Just my 2p worth.
