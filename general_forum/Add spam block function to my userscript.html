Hello everyone,

I was annoyed with the spam posts in the forum, so I added a function to hide them to my userscript for this site.

It can be found here: http://github.com/andreasmaurer/markforster.net-improvements-userscript/

If you have my userscript installed, it should update automatically, otherwise you can just re-install it.

To use it, check the checkbox "Hide posts I mark as spam" in the sidebar. This inserts a "Mark as spam" button to every topic in the forum overview. Once clicked, the topic will be hidden.

If you accidentally marked a topic as spam, uncheck the checkbox "Hide posts I mark as spam" in the sidebar, then click "Mark as not spam" for that topic.

It's not a perfect solution since it doesn't really remove the spam posts, but it's the best I can do without access to the admin panel of the site.
