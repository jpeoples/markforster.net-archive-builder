Regarding the existing group, I think part of the original intention was to open the discussion up to a wider audience and in a different format than the main forum (here).

But I think Mark's new goal is to discuss and strategize how to *create* sites like that.

I could easily envision that this new proposed Super Hero Strategizers and World Conquerers group, would eventually try, as part of their evangelizing efforts, to create a Yahoo Group to discuss the tool and how to use it.

So, if we change the AutoFocusTM group to become the World Conquerers group, here is what would happen:
(1) People on that group already who aren't interested in World Conquering would drop off and/or fade away.
(2) Eventually the World Conquerers would decide they want a new Yahoo Group as a discussion forum for what AF is, how to use it, and so on; such a forum would help spread the word and perhaps help make Mark some money
(3) The World Conquerers would thus create this new Yahoo Group
(4) The World Conquerers would need to bring in new people (some of whom perhaps have already left the existing Yahoo Group)

So, I think this might be more effective:
(1) Create a new World Conquerers group on Yahoo - great venue for the discussion. Plus, Mark could create it himself and be the owner, so he could control access (not sure a group like that would want public access). He could assign moderators if he wanted.
(2) Let the existing AutoFocusTM group continue as is, as a public forum to discuss the benefits and uses of AF

Anyway, just some ideas, maybe I'm totally missing the point here... :-)
