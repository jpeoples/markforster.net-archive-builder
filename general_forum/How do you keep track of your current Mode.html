Firstly, thanks to Mark for the revised instructions. I've found them immensely useful.
Many a times, during the day, I tend to forget which mode I'm in. If I'm doing a series of short-duration tasks, I remember the mode. But, when I come to a task which takes a long time, or when I go on a break or to a long meeting and come back to the list, I don't remember which mode I'm currently operating in: reverse or forward.
Has anybody experienced this? If so, how do you keep track of your current mode?

Reply by Andreas Hofmann

2009-07-31 8:11
My spontaneous answer would be "when in doubt, reverse mode". But this would be a problem if you strike the "Change to Forward Mode" task from the list before entering forward mode and re-add it only when you leave forward mode. But then the existance of the "Change to Forward Mode" task itself would be telling in which mode you are.

Perhaps you could use the same technique Mark advocates for marking the currently active task. When you enter forward mode, make a dot beside the "Change to Forward Mode" task, when you leave forward mode, make a line out of the dot. So if you don't know in which mode you're in, search for the "Change to Forward Mode" task. If there's a dot, you're in forward mode, else you're in reverse mode.
