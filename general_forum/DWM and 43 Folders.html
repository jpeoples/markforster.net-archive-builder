I've been using both these systems for several months at least, and it was only today that it clicked how congruent they are. So from now, instead of having lines with the calendar dates, I am using dividing lines marked 1 through 31 corresponding to the card dividers in a 43 folders system. For example today, in 43 folders, I moved the 1 divider behind the 31 divider having dealt with the days papers, while in my DWM file, after deleting the tasks which dropped off today, I dragged the 1 dividing line down below the 31 line.

To work this way, you need to be using the original DWM system, in a computer file, and using the 43 Folders tickler/filing system.
