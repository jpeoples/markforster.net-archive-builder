2011-03-24 10:14
@Sarah: I'm experimenting using Evernote for SF3, I have one note, then a page break for each page and column, eg
Page 01.C1 Dismissed for review tasks
Page 02.C1 Open tasks: these are numbered up to a maximum of 20 tasks, then I start a new page.
Page 02.C2 Unfinished/urgent tasks: these are not numbered, each has a check-box
Page 03.C1 Open tasks
Page 03.C2 Unfinished/urgent tasks
and so on.

The current pages - C1 and C2 - have bold text headers. Finished open tasks on C1 pages have a strike-through when done, C2 tasks get check box ticked.

There is a separate notebook for projects whose tasks I can pull across to the relevant C1 or C2 as necessary, or maybe not! However, as someone has already mentioned, I could have the projects within the SF3 notebook, so I've dragged-and-dropped a project note into the SF3 notebook, works nicely.

I've also gone for Evernote (probably going to drop MS OneNote) as it is one of the few applications to include Windows Mobile 6.5, my current smartphone. I will install and test this shortly, and also on my wife's Galaxy Tablet (Android). I like the concept of having the information available on more than one device.

However, I am also intrigued by your use of Toodeldo which is not available for WM nor do I have an iPhone, but I may get an Android smartphone later this year.

I stopped using GTD and MLO some years ago (ok, when Mark started his first incarnation of AF in December 2008!) as I did not like the overhead.
