I'm using ListPro, and my Office AutoFocus file now has 12 inactive lists and 8 active ones. Clearly as time goes on the number of inactive lists will grow and grow. Which makes ListPro get incrementally slower. Especially on the PDA! So it makes sense to move them into a separate archive file. I have a dismissed item list/folder. But one thing I'm doing helps me see my progress. I keep my original pages archived (at least for now) so if I need to check whether I've done something, I can easily search and find the completed item. So I started with page 1. Then when that page was finished and I reached my self-created limit of 25 pages, I created Page 1*. That tells me it's the second generation of page 1. I will probably create Page 1** soon. At some point, I will archive all the successive generations and will go back to regular Page 1. I use this method so I don't have Page 50-Page 70 and so on.
