I like using MindManager for tasks because it shows relationships. I have a bit of trouble with lists and they are WAY easier than block text for me to manage ;-)

Laying projects out in MM gives me a visual view of how things relate. So I have something that is a cross between a concept map and a PERT chart. IOW, as I move toward the edges of the map I get to the things I should be working on that will enable the ones closer to the center to be achieved. (The center being Nirvana, of course ;-)

The "problem" with the MM approach is that it works better for projects and is less useful in handling the one-of things that clutter our lives. Also, it does not have any kind of AutoFoucs attributes ... IOW, it orients toward Project and not Time or Level of Effort or simply how long stuff is hanging around not getting done. What it adds to my life is that I can see those relationships easier and AF robs me of that dimension.

For example: on my AF list I have sprinkled various tasks about maintaining several home computers in various stages of disrepair. But some tasks are common to some (or all) machines and some tasks depend on other, non computer related housework. (I need to reorganize my office if I plan to put a computer system in there, for example.) Laying it out in MM I can SEE that there is little point building my office system first because then I'd be off in reorganizing my office shelving and ... well, a whole lot of things I'm not willing to do first. But I can also SEE that if I solve a problem with organizing my NAS and USB storage devices so they work over the network, I'll solve multiple problems on multiple computers with the least effort.

Another effect of using MM is that it provides some motivation because it shows how one thing may be affecting other things. IOW if you want to do more exercise you need a place to do that so you might want to clear an area in the living room or some place. And if you want to put in a new bookcase you might also want it in the living room. Etc. So you can see, visually, that reorganizing the living room is a bottleneck for several things you'd like to achieve. So a simple line item in AF such as "Plan New Living Room Utilization" appears now as a bunch of arrows which SHOW exactly WHY you want to do it ... no words needed at all.

But my actual implementation is more dealing with projects. I don't work from MM for everything. For one thing, it would take too long. ;-)
