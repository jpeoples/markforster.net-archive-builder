If you have Outlook, you can easily set up the Task reminders to do just what you want. For the first occurrence just set a task for 15 mins from the current time (although the drop down menu doesn't give a 15 min option, you can type the time in manually). Then when the alarm goes off, put it on 15 mins snooze. You can continue doing that all day.

P.S, It's only fair to say that I use a Time Timer myself (the physical version) myself and love it. I haven't tried the software version though.
