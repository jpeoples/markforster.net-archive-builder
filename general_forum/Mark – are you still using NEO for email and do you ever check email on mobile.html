Troy:

1. I don't use NEO any longer, but only because I found the program kept seizing up. Probably because it's dealing with a huge amount of past email. NEO indexes the Achive folder along with any active folders. With the much smaller volume of email I have at the moment I can cope perfectly well with Outlook on its own.

2. I only use my Smart Phone for previewing email while I'm away from my desk - I only answer an email on the Phone if it is really urgent. I delete all email from the Phone once I've read it. In order to do this, it's very important to make sure that the Phone is set NOT to delete email from the server so that it's still available for Outlook to download. I have Outlook on my computer set to delete email from the server once downloaded, so the Phone only downloads what I haven't already seen.
