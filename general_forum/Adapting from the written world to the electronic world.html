Use Outlook tasks

Setup:
--------
Table view
Allow editing in table
Show columns
completed checkbox
current task (user defined yes/no) Or use a category flag.
Subject
Start date
Sort by "modified date"
Filter by "completed" ="No" (optional. You can set other views)
group by "start" (low to high)

Operate:
----------
Start at the current flagged task
When you have finished, either
- click the "completed" checkbox OR
- tab to the start date and type d or now
- unflag the current flag if it is still live
Cycle through the group, clicking on each task subject until you see one to do (then flag it current and do it, as above)

To enter a task, type subject, tab to start date, type "d" for tomorrow's date or "now" for today's date.
At the start of the day, type the line and "now"

Works fine. Much simpler than it looks
